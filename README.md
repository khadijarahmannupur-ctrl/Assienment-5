## 1️⃣ What is the difference between var, let, and const?
Ans: Difference between var, let, and const:

var, let, and const are used to declare variables in JavaScript.

1.var is function-scoped, can be redeclared and reassigned, and it is hoisted with undefined.

2.let is block-scoped, cannot be redeclared in the same scope, but it can be reassigned.

3.const is also block-scoped, but it cannot be reassigned or redeclared, and it must be initialized when declared.

## 2️⃣ What is the spread operator (...)?
Ans: Spread Operator (...) in JavaScript is used to expand or spread the elements of an array or the properties of an object into another array or object.

It is commonly used to copy arrays.

## 3️⃣ What is the difference between map(), filter(), and forEach()? 
Ans: map(), filter(), and forEach() are methods used to work with array elements.

1.forEach() runs a function for every element in an array. It does not return a new array.

2.map() runs a function for every element and returns a new array with changed values.

3.filter() checks a condition and returns a new array with elements that match the condition.

## 4️⃣ What is an arrow function?
Ans: An arrow function is a shorter way to write a function in JavaScript.
It was introduced in ES6 (2015) and uses the () => syntax.

It helps to write clean and shorter code.

## 5️⃣ What are template literals?
Ans: Template literals are a modern way to write strings in JavaScript.
They use backticks (``) instead of single (' ') or double (" ") quotes.

They allow us to insert variables inside a string using ${}.
